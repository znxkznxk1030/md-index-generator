# Markdown Index Generator

## Markdown Index Generator 예시

### 예시 1

#### 입력 input-001.md

```md
# Header 1

## Title 1

### sub-title 1

##2 Title 2

### sub-title 2

## Title 3

### sub-title 3
```

#### 출력

```md
- [Header 1](#header 1)
  - [Title 1](#title 1)
    - [sub-title 1](#sub-title 1)
    - [sub-title 2](#sub-title 2)
  - [Title 3](#title 3)
    - [sub-title 3](#sub-title 3)
```
